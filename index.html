Coin Tsunami Cryptocurrency Price Tracker

This is a Cryptocurrency price tracker using a cryptocurrency price API. The frontend is written in React while the backend uses Express. Users are able to see the latest market data, create an account, watch and compare their favorite coins, as well as read the latest cryptocurrency news. Languages and Frameworks: HTML, CSS, Javascript, ReactJs, NodeJs, and express.

Trivia Game

I coded a Trivia game in React.js for an apprenticeship application and it turned out pretty cool! You can check out the code, clone the repo and play the game here. There are some pretty cool questions!

Facial Recognition Application

A code-along tutorial creating a web application from scratch. You can create a user, sign-in, and then submit photo url’s for the AI to idenitify faces. For now, it only does one face at a time. Try it out here. Languages and Frameworks: HTML, CSS, Javascript, ReactJs, NodeJs, and express.

ft_printf

This project is my recreation of the printf command with custom flags. I grew a lot as a programmer during this project, working with various data structures and programming concepts to bring the command to life.

Libft 2.0
